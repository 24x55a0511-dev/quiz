# 🧠 AI Quiz Generator Platform

A production-level **AI-powered Quiz Generation Web Application** that generates high-quality MCQ quizzes on any topic using Generative AI (Groq API) with advanced UI/UX, difficulty modes, analytics, progress tracking, history, and theme customization.

## 🚀 Project Overview

This application allows users to instantly generate quizzes from any topic such as:

- Programming Languages (JavaScript, Python, Java, C++)
- Frameworks (React, Next.js, Node.js)
- Tools (Git, Docker, Kubernetes)
- Skills (Communication, Marketing, Finance)
- Academic topics (Math, Physics, Biology)
- Any custom topic entered by the user

The system generates **random MCQ questions** based on the selected difficulty and number of questions, then provides results, analytics, and progress tracking.
